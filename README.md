PhotonIDE is an **open source and extensible IDE** built in Python, designed to be modern, flexible, and modular.

## Setup

### Clone the repository

```bash
git clone https://github.com/mstchristian/PhotonIDE.git
cd PhotonIDE
```

### Create a virtual environment

```bash
python -m venv .venv
```

### Activate the virtual environment

```bash
# Windows (cmd):
.venv\Scripts\activate

# Windows (PowerShell):
.venv\Scripts\Activate.ps1

# macOS / Linux:
source .venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run PhotonIDE

```bash
python main.py
```
